Different Selectors in CSS

Element Selectors

selectors target HTML tags by name. Use them when you want the same style for every instance of an HTML tag.
Example sentence for the page

“Element selectors apply CSS rules to all elements of a given HTML tag. For example, the selector p styles every paragraph on the page.”

Descendant selectors

A descendant selector is used in CSS to style elements that are inside (nested within) other elements

Attribute Selectors

Attribute selectors style elements that have a certain attribute or a specific attribute value.
Syntax varies depending on what you want to select.

The selector [placeholder] selects all elements that have a placeholder attribute, regardless of its value.

The selector [type="email"] selects all elements whose attribute exactly matches the value.
